# 个人品牌官网搭建完成

## 成果概览

为柳秉含创建了基于 OPC 利基定位的个人品牌门户网站，定位：**AI 加速的测试交付服务专家**。

## 网站结构

| 板块 | 内容 |
|---|---|
| **Hero 主视觉** | 姓名 + 头衔 + 软件评测师证书展示 + 7/3/1 核心数据 |
| **核心服务** | 测试报告与验收（推荐）、测试基建搭建、信息无障碍测试 |
| **核心资质** | 持证 / 3次基建 / 无障碍 / AI 加速 四大卖点 |
| **差异化优势** | 4 条差异化理由 + 服务过的项目类型标签云 |
| **职业经历** | 2017-2026 时间线，从「唯一测试」到「一人交付」 |
| **CTA** | 紧急需求引导 |
| **联系方式** | 杭州 / 微信 / 邮箱 / 需求表单 |

## 技术方案

- 纯静态 HTML 单页站，响应式设计
- 内联 CSS/JS，无需构建工具
- 深色 Hero + 浅色内容的专业商务风格
