## 如何修改内容

### 1. 联系方式
在 `index.html` 中找到「直接联系我」部分，修改以下信息：
- 微信：自行添加微信号（当前留空，建议添加）
- 邮箱：已填入 `liubinghan@xjagent.cn`，如需修改请搜索替换

### 2. 证书图片
当前使用 `certificate.png`，如需更换：
1. 删除或替换 `certificate.png`
2. 如需修改文件名，同步修改 `index.html` 中 `<img src="certificate.png">` 的引用

### 3. 服务内容与定价
在「核心服务」板块（`#services`）中直接编辑文字内容即可。

### 4. 颜色/风格
所有样式为内联 CSS，在 `<style>` 标签内修改：
- 主色：`--primary: #1a365d`
- 强调色：`--accent: #d69e2e`

## 部署方式

### 方案 A：静态托管（推荐，免费）
- **GitHub Pages**：推送到 GitHub 仓库，开启 Pages 即可
- **Cloudflare Pages**：拖拽上传文件夹
- **Vercel / Netlify**：导入 Git 仓库自动部署
- **Gitee Pages**（国内访问快）：国内用户的备选

### 方案 B：自有服务器
将 `personal-brand-site/` 目录下所有文件上传到服务器任意目录，配置 nginx/Apache 指向该目录即可。

### 方案 C：对象存储 + CDN
- 阿里云 OSS / 腾讯云 COS / AWS S3
- 开启静态网站托管 + CDN 加速

## 注意事项
1. **备案问题**：当前域名 `api.xjagent.cn` 备案未通过。如需使用国内域名，建议先使用香港/海外服务器 + 未备案域名，或走方案 A 的海外平台。
2. **表单功能**：当前联系表单为静态展示，提交后会提示用户直接微信/邮件联系。如需真正可用的表单，可接入：
   - 腾讯云 CloudBase 云函数（免费额度）
   - 第三方表单服务（如金数据、麦客）
   - 直接放微信二维码图片替代表单
3. **证书隐私**：网站展示证书时，已对敏感信息（证件号码、管理号）做了模糊处理，如需进一步处理可用图片编辑工具打码后替换 `certificate.png`。
