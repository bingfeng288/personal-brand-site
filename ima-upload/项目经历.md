## 项目一：Claw —— AI 时尚电商微信小程序

### 项目概述

- **项目类型**：AI 虚拟试穿 + AI 个性化推荐微信小程序
- **目标**：快速上线获取用户，验证 AI 时尚电商闭环
- **角色**：全栈开发（前端 + 后端 + DevOps）
- **状态**：开发中

### 技术架构

| 层级     | 技术栈                                              |
| -------- | ----------------------------------------------------- |
| 前端     | 微信小程序原生框架（WXML + WXSS + JS）                |
| 后端     | FastAPI（Python）                                     |
| 数据库   | MariaDB + Redis                                       |
| 部署     | Docker Compose + nginx                                |
| 服务器   | 阿里云（已到期，计划迁移）                            |
| API 地址 | `api.xjagent.cn`（已配置 SSL）                        |
| 架构模式 | 多 Agent 协作架构（规划 Agent + 执行 Agent + 校验 Agent），基于 AgentScope |

### 开发规范与约束

- **WXML 严格约束**：禁用 JS 方法（如 `toFixed`）、可选链操作符、点号语法
- **插值修复**：`{{ }}` 插值需通过嵌套 `<text>` 元素修复显示问题
- **构建优化**：已配置 `minifyWXML: true` + `lazyCodeLoading: true`
- **编码流程**：编码前必须从远程拉取代码；基础代码就绪后再进行功能开发

### 已完成工作

- [x] 全栈框架搭建（FastAPI + MariaDB + Redis + Docker Compose + nginx）
- [x] API 上线阿里云（api.xjagent.cn，已配置 SSL）

- [x] 小程序页面模块梳理（AI 推荐、关于、地址、优惠券、帮助、设置等）

## 项目三：个人品牌官网（OPC 信任锚点）

### 项目概述

- **目标**：建立个人专业形象，作为一人公司（OPC）的信任锚点
- **技术**：静态 HTML 单页，内联 CSS/JS，无后端依赖
- **部署**：EdgeOne Pages + GitHub Pages（gh-pages 分支 + GitHub Actions）
- **状态**：已完成初版

### 网站结构

| 区块       | 内容                                               |
| ---------- | ---------------------------------------------------- |
| Hero       | 姓名、职位、证书展示（软件评测师）                   |
| 服务卡片   | 测试报告服务 / 测试基建搭建 / 无障碍测试            |
| 资质展示   | 软件评测师证书图片                                   |
| 职业时间线 | 7 年测试经验时间线                                  |
| CTA        | 联系表单（mailto 方式，兼容国内网络环境）            |
| 联系方式   | 手机（可拨打）/ 微信（可复制）/ 邮箱 / 企业微信二维码 / 飞书二维码 / 钉钉二维码 |

### 技术要点

- 响应式布局，移动端适配
- 证书图片压缩后内联展示（360KB）
- 联系表单使用 `mailto:` 兼容国内环境（FormSubmit.co 被墙）
- 企业微信 / 飞书 / 钉钉二维码并列展示
